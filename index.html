Stack

Las siete landings comparten un stack deliberadamente minimalista. Sin frameworks. Sin build step. Solo lo necesario para que cualquier navegador moderno las renderice instantáneamente.

  • HTML5Semántico, accesible, español.
  • CSS3Custom properties + grid/flex.
  • JS vanillaSin librerías. Defer + IntersectionObserver.
  • Google FontsDisplay + sans/mono por sitio.
  • Unsplash CDNImágenes con licencia abierta.
  • GitHub PagesDespliegue gratuito en CDN global.

Proceso

  1. 01

    Identidad

    Marca ficticia, slug, paleta de 5 tonos, par tipográfico y mood en una sola página de brief.

  2. 02

    Arquitectura

    6-8 secciones mínimas: nav, hero, destinos, narrativa, testimonio, footer. Variantes opcionales por tema.

  3. 03

    Construcción

    HTML semántico, CSS con tokens y JS vanilla. Responsive 980/720, accesibilidad básica, SEO mínimo.

  4. 04

    Despliegue

    Repo individual en GitHub, Pages activadas, URL pública en minutos. Cero infra propia.
