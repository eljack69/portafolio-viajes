# Portafolio — 7 landings de viajes

Galería estática que agrupa **7 landings** de agencias de viajes ficticias,
cada una con identidad visual, tipográfica y arquitectura propia.

## URL pública
- https://eljack69.github.io/portafolio-viajes/

## Stack del portafolio (igual que cada sitio)
HTML5 + CSS3 (custom properties + grid) + JS vanilla.
Sin build, sin dependencias. Google Fonts + Unsplash CDN.
Despliegue en GitHub Pages.

## Aviso
Marcas, datos de contacto, testimonios y números son **ficticios**.
Imágenes vienen del CDN público de Unsplash con licencia abierta;
reemplazar por fotografía propia antes de cualquier uso comercial.
